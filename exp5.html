Aim- To design a responsive web page using Bootstrap framework components such as grid system, cards, and buttons. Objective- To understand the basics of Bootstrap framework. To learn how to use the Bootstrap grid system for layout design. To apply pre-defined Bootstrap classes for styling buttons and cards. To create a responsive web page that adapts to different screen sizes. Software Requirement- Text Editor (Notepad / Visual Studio Code / Sublime Text) Web Browser (Google Chrome / Mozilla Firefox / Microsoft Edge) Internet Connection (for Bootstrap CDN) Algorithm: 1. Open a text editor. 2. Create a new HTML file. 3. Write the basic HTML structure. 4. Include Bootstrap using CDN links. 5. Create a responsive layout using the Bootstrap grid system. 6. Add Bootstrap components such as buttons and cards. 7. Save the file and open it in a web browser. 8. Resize the browser window to verify responsiveness Theory- Bootstrap is a popular front-end framework used for developing responsive and mobile-first websites. It provides ready-made CSS classes that simplify web design. In this program: Grid System: Bootstrap uses a 12-column grid system. container is used to hold the content with proper spacing. row is used to create a horizontal row. col-md-4 divides the row into 3 equal columns (4+4+4 = 12). Cards: The card class is used to create content boxes with padding and styling. It is commonly used for grouping related content. Buttons: Bootstrap provides predefined button classes like: btn-primary (blue) btn-success (green) btn-danger (red) These classes help in quickly designing attractive UI without writing custom CSS. Conclusion- Thus, we successfully created a responsive web page using Bootstrap. This program helped in understanding grid layout, cards, and button styling using Bootstrap classes.
