Aim- To create a web page using JavaScript that validates form input fields such as User ID, Contact number, and Password. Objective- To understand form validation using JavaScript. Software Requirement- Text Editor (Notepad / Visual Studio Code / Sublime Text) Web Browser (Google Chrome / Mozilla Firefox / Microsoft Edge) Algorithm: 1. Open a text editor. 2. Create a new HTML file. 3. Write the basic HTML structure. 4. Create a form with input fields. 5. Write JavaScript code for form validation. 6. Link the JavaScript code to the form using the on submit event. 7. Save the file and open it in a web browser. 8. Enter valid and invalid inputs to verify validation. Theory- JavaScript is used to make web pages interactive and dynamic. One important use of JavaScript is form validation, which ensures that users enter correct data before submission. In this program: getElementById() is used to access form input values. The if condition checks whether any field is empty. The contact number is validated to ensure it has exactly 10 digits. isNaN() is used to check whether the input is a valid number. Password and Confirm Password fields are compared to ensure they match. The function is called using onsubmit event, and form submission is prevented if validation fails. This improves data accuracy and prevents incorrect input submission. Conclusion- Thus, we successfully implemented form validation using JavaScript. This program helped in understanding how to validate user input and control form submission using client-side scripting.
