Theory- JavaScript is used to make web pages interactive and dynamic. One important use of JavaScript is form validation, which ensures that users enter correct data before submission. In this program: getElementById() is used to access form input values. The if condition checks whether any field is empty. The contact number is validated to ensure it has exactly 10 digits. isNaN() is used to check whether the input is a valid number. Password and Confirm Password fields are compared to ensure they match. The function is called using onsubmit event, and form submission is prevented if validation fails. This improves data accuracy and prevents incorrect input submission. Conclusion- Thus, we successfully implemented form validation using JavaScript. This program helped in understanding how to validate user input and control form submission using client-side scripting.
